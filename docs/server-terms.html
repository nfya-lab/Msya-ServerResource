    はじめに

    この利用規約(以下「本規約」といいます)は、本サーバーの利用条件を定めるものです。

    本サーバーをご利用される全てのユーザーの皆様(以下「ユーザー」といいます)には、本規約に従って本サーバーをご利用いただきます。

    第1条(適用)

    本規約は、ユーザーと本サーバー運営者との間の本サーバーの利用に関わる一切の関係に適用されます。

    第2条(禁止事項)

    ユーザーは、本サーバーの利用にあたり、以下の行為をしてはなりません。

  • 法令または公序良俗に違反する行為
  • 犯罪行為に関連する行為
  • 倫理的または社会的によくない行為
  • 差別的・攻撃的な発言や行為
  • 本サーバーの運営を妨害するおそれのある行為
  • 他のユーザーに対する嫌がらせや誹謗中傷
  • チートツールやマクロの使用
  • バグやグリッチの悪用
  • サーバーに負荷をかける行為
  • 他のユーザーのアカウントを無断で使用する行為
  • その他、運営者が不適切と判断する行為

    • 第3条(ゲームプレイに関するルール)

    フェアプレイの徹底

  • マクロ・チートの使用は一切禁止です
  • 自動化ツールやクリック補助ツールの使用は禁止です
  • 不正な方法でゲームを有利に進めることは禁止です

    • 放置プレイについて

  • ゲーム中の放置は禁止ではありませんが、以下のことが起こります
  • 放置していると自動的にキックされる場合があります
  • 他のプレイヤーに倒される・おもちゃにされても文句は言えません
  • ポイントを失ったり不利益を被っても、運営は一切関知しません
  • 放置は自己責任です

    • 妨害行為の禁止

  • 悪質な邪魔行為は禁止です
  • 味方への意図的な妨害行為
  • ゲーム進行を故意に遅らせる行為
  • 他のプレイヤーがゲームを楽しめないようにする行為
  • ロビーでのゲーム開始妨害

    • 第4条(各ミニゲームのルール)

    各ミニゲームには個別のルールがあります。必ず確認し、遵守してください。

  • ゲーム開始前に表示されるルールを必ず読むこと
  • ゲームモードごとの禁止事項を守ること
  • チーミング許可の有無など、特別なルールに従うこと
  • ルールが分からない場合は、事前に確認すること

    • 第5条(コミュニケーション)

  • チャットでは敬意を持った言葉遣いを心がけること
  • スパムや連投は禁止です
  • 個人情報の公開や要求は禁止です
  • 政治・宗教に関する議論は避けてください
  • ゲーム中の過度な煽り行為は禁止です

    • 第6条(ポイント・報酬)

  • 不正な方法でポイントを獲得することは禁止です
  • 複数アカウントを使用した不正は永久BANの対象です

    • 重要事項

    喧嘩両成敗です

    トラブルが発生した場合、理由を問わず関わった全員が処罰の対象となります。

    本規約に違反した場合、警告、一時的なBANまたは永久BANなどの措置を取る場合があります。

    悪質な場合は、警告なしに即時BANとなることもあります。

    マクロ・チートの使用が確認された場合は厳しく対処します。

    第7条(免責事項)

    本サーバーの利用により生じたいかなる損害についても、運営者は一切の責任を負いません。

    サーバーのデータが失われた場合、基本的にロールバックや補償は行いませんが、運営者の判断により対応する場合があります。

    第8条(サーバーのメンテナンス)

    運営者は、事前の予告なくサーバーのメンテナンスやアップデートを行うことがあります。

    この際、一時的にサーバーが利用できなくなる場合があります。

    第9条(規約の変更)

    運営者は、必要と判断した場合には、ユーザーに通知することなくいつでも本規約を変更することができます。

    変更後の本規約は、本サーバー上に表示した時点より効力を生じるものとします。

    第10条(お問い合わせ)

    本規約に関するお問い合わせは、Discordサーバーのお問い合わせまでご連絡ください。
